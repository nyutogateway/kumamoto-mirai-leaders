# 熊本未来リーダーズ WordPressテーマ

静的サイト（熊本未来リーダーズ）をWordPress化したクラシックテーマです。
リーダー（インタビュー）はカスタム投稿タイプ `leader` ＋ ACF で管理します。

---

## 1. 必要環境
- WordPress 6.0 以上 / PHP 7.4 以上
- プラグイン **Advanced Custom Fields (ACF)**
  - **無料版**: ヒーロー項目（英語名・会社名・役職・リード文・URL・ロゴ）が使えます。記事本文は投稿本文エディタで書きます。
  - **ACF PRO**（推奨）: 上記に加え、**MY STORY / Q&A（リピーター）** と **ギャラリー** が管理画面で編集できます。

## 2. インストール
1. このフォルダ（`kumamoto-mirai-leaders`）をzip化するか、そのまま `wp-content/themes/` にアップロード。
   - 管理画面から入れる場合: **外観 → テーマ → 新規追加 → テーマのアップロード** で zip を選択。
2. **外観 → テーマ** で「熊本未来リーダーズ」を有効化。
3. **プラグイン** で Advanced Custom Fields を追加・有効化（PRO推奨）。
   - ACFのフィールド定義はテーマ同梱（`acf-json/`）で**自動読込**されます。インポート作業は不要です。
   - もし反映されない場合は **カスタムフィールド → 同期** から取り込んでください。

## 3. 初期設定
- **リーダー**メニューが管理画面に出ます（人物アイコン）。
- 固定ページを作成:
  - お問い合わせ … スラッグ **`contact`**（テンプレート `page-contact.php` が自動適用）
  - プライバシーポリシー … スラッグ **`privacy`**（本文にポリシー文を貼り付け）
- メニュー（**外観 → メニュー**）を作成し、位置「ヘッダーナビ」「フッターナビ」に割当。
  - 未設定でも CONCEPT / LEADERS / CONTACT / POLICY のフォールバックが出ます。
- トップページは `front-page.php` が自動的に使われます（表示設定の変更は不要）。

## 4. リーダー（インタビュー）の追加
**リーダー → 新規追加** で以下を入力:

| 項目 | 入力先 |
|---|---|
| 氏名（例: 宮本 功） | タイトル |
| 人物写真（カード・記事の顔写真） | **アイキャッチ画像** |
| 英語名（ISAO MIYAMOTO） | ACF「英語名」 |
| インタビュータイトル（記事の大見出し） | ACF「インタビュータイトル」 |
| 会社名 / 役職 | ACF「会社名」「役職」 |
| リード文（ヒーロー説明） | ACF「リード文」 |
| 会社URL | ACF「会社URL」 |
| 会社ロゴ | ACF「会社ロゴ画像」 |
| 本文（MY STORY / Q&A） | ACF「MY STORY」リピーター（PRO）。無料版は投稿本文に記述 |
| ギャラリー | ACF「ギャラリー」（PRO） |
| 締めメッセージ | ACF「締めのメッセージ」 |

- **表示順**は「ページ属性 → 順序」（数字が小さいほど先頭）で調整。
- **カード色**（青→ピンク→黄→ティール）は並び順に応じて自動で循環します。
- 5枚ごとに熊本の名所名物スライダーが自動で挿入されます。

## 5. 補足・今後の調整ポイント
- **OGP画像**: `header.php` では `wp_head()` に任せています。SNSサムネを厳密に出すなら Yoast SEO / SEO SIMPLE PACK 等のSEOプラグイン導入を推奨。
- **お問い合わせフォーム**: 現状はデザイン用の静的フォームです。実送信には **Contact Form 7** 等を導入し、`page-contact.php` の `<form>` をショートコードに置き換えるか、本文にCF7ショートコードを貼ってください（`.cform` クラスで意匠を流用可）。
- **アニメーション/演出**（reveal・カード遷移マスク・画像追従・スライダー自動スクロール）は元サイトのJS/CSSをそのまま同梱しています。

## ディレクトリ構成
```
kumamoto-mirai-leaders/
├── style.css              テーマ情報
├── functions.php          CSS/JS読込・CPT・ACF連携・ヘルパー
├── header.php / footer.php
├── front-page.php         トップ（FV/CONCEPT/LEADERSループ/FM791）
├── single-leader.php      インタビュー記事
├── page.php               汎用固定ページ
├── page-contact.php       お問い合わせ
├── index.php              フォールバック
├── inc/cpt.php            カスタム投稿タイプ leader
├── acf-json/              ACFフィールド定義（自動読込）
├── template-parts/        名所スライダー
└── assets/ css / js / img アセット一式
```
